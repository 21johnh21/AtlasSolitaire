How to Play

If you know classic solitaire (Klondike), you already know how to play! The twist: instead of organizing cards by hearts, diamonds, clubs, and spades, you'll match cards from the same geographic group.

For example, one game might feature Asian countries, European countries, African countries, and American countries. Build sequences in the tableau and complete foundation piles by matching geographic groups.
