### Promotional Text Best Practices:
- Update seasonally to keep listing fresh
- Test different angles (gameplay, competition, relaxation)
- Change it when you add new features
- A/B test if you have multiple regions

### Description Best Practices:
- Front-load important info (first 3 lines visible before "more")
- Use keywords naturally (helps ASO but don't stuff)
- Break into scannable sections
- Include clear calls-to-action
- Mention free/no ads upfront if applicable

### Keyword Best Practices:
- Don't repeat words (Apple indexes each word once)
- Use singular form (Apple matches plurals automatically)
- Avoid spaces after commas (wastes characters)
- Research competitor keywords in App Store
- Update based on search term performance
